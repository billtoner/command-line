# Text Processing

Filtering, transforming, and reshaping structured and unstructured text.

- [jq](../../tool-notes/jq.md) — command-line JSON processor; filter, transform, and reshape JSON
- [yq](../../tool-notes/yq.md) — YAML/JSON/XML processor with jq-style syntax; preserves YAML comments on in-place edits
- [jc](../../tool-notes/jc.md) — convert command output (ps, df, dig, lsblk, ifconfig…) to JSON for jq pipelines
- [gron](../../tool-notes/gron.md) — flatten JSON into greppable assignments; reversible with `gron -u`
