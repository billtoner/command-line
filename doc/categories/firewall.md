# Firewall

Packet filtering and NAT — both the legacy `iptables` interface and modern `nftables`, plus the friendly Ubuntu frontend.

- [iptables](../../tool-notes/iptables.md) — classic packet filter; rules, chains, NAT, conntrack, rate limiting
- [nft](../../tool-notes/nft.md) — modern nftables CLI; sets, named tables, atomic loads, live monitoring
- [ufw](../../tool-notes/ufw.md) — Ubuntu's friendly wrapper; named services, app profiles, rate-limit shortcut
