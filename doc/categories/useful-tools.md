# Useful Tools

Miscellaneous tools that don't fit a tighter category.

- [atuin](../../tool-notes/atuin.md) — TUI fuzzy-search shell history backed by SQLite
- [sqlite3](../../tool-notes/sqlite3.md) — official SQLite CLI; schema, queries, CSV import/export, pragmas
- [sqlite-utils](../../tool-notes/sqlite-utils.md) — CLI for querying, dumping, and importing SQLite data
- [DB Browser for SQLite](../../tool-notes/db-browser-for-sqlite.md) — GUI for inspecting and editing SQLite databases
- [vd](../../tool-notes/vd.md) — VisiData TUI spreadsheet; opens CSV/JSON/SQLite/Parquet, sort/filter/pivot, save as anything
- [grip](../../tool-notes/grip.md) — render local markdown via GitHub's API and serve on localhost; identical to github.com
